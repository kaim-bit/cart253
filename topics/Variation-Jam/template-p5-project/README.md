# Variation Jam

Kai Maquivar

[View this project online](URL_FOR_THE_RUNNING_PROJECT)

## Description

this game has 3 games inside of it inspired by rock paper scissorts, the rock game is rock paper scissors, the paper game is a math test with funny questions randomly generated, and the third game is a collection game where scissors is the player and they try to eat normal paper while avoiding evil paper.
